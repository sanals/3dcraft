STEP 1: GUARANTEED WORKING SOLUTION (DO THIS FIRST)
================================================================================

A) CLEAR BROWSER CACHE AND RESTART

   1. Stop the dev server (press Ctrl+C in terminal)
   
   2. Clear browser cache:
      - Press Ctrl+Shift+Delete (or Cmd+Shift+Delete on Mac)
      - Select "All time"
      - Check: Cookies, Cache, Cached images and files, Local Storage
      - Click "Clear data"
   
   3. Restart dev server:
      cd /vercel/share/v0-project
      npm run dev
   
   4. In your BROWSER:
      - Close browser completely (all windows)
      - Open fresh browser
      - Go to http://localhost:3000
      - Press Ctrl+Shift+R (hard refresh - NOT Ctrl+R)
      - WAIT 10-15 seconds for full load
      - You should now see the BLUE GRID

B) IF THAT DOESN'T WORK:

   Try the automated setup script instead:
   
   Mac/Linux:
   cd /vercel/share/v0-project
   ./setup.sh
   
   Windows:
   cd C:\path\to\voxel-project
   setup.bat

STEP 4: IF STILL NOT WORKING
================================================================================

A) Check browser console for errors:
   - Press F12 (open Developer Tools)
   - Go to Console tab
   - Look for red error messages
   - Screenshot any errors
   - See TROUBLESHOOTING.md for solutions

B) Run diagnostic command:
   
   Open browser console (F12) and paste:
   
   const canvas = document.querySelector('canvas');
   console.log({
     canvasExists: !!canvas,
     width: canvas?.clientWidth,
     height: canvas?.clientHeight,
     webGL: !!canvas?.getContext('webgl')
   });
   
   Expected output: width and height should be large numbers (1200+, 800+)
                    NOT 0

C) If canvas width/height are 0:
   - This means sizing issue (old cache)
   - Go back to STEP 1 and be more thorough
   - Clear browser cache from browser settings too

D) If console shows errors:
   - Copy the errors
   - See TROUBLESHOOTING.md for solutions by error type
